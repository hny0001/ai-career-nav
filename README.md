# AI Career Navigator (AI 职业导航) 🧭

欢迎来到 **AI Career Navigator (AI 职业导航)** 项目！这是一个专为 AI 时代设计的职业发展咨询导航站点，旨在帮助广大职场人士、开发者和各行各业的从业者评估所在岗位在人工智能浪潮下的替代风险，并提供系统化的转型建议与职业规划策略。

## 🌟 核心功能

1. **职业进化史科普 (Career Evolution Timeline)** ⏳
   梳理了从农业文明到 AI 智能时代的技术与生产力演变历程，深刻剖析每一次技术革命为社会和个体带来的颠覆性影响。

2. **AI 替代风险热力图 (AI Risk Heatmap Matrix)** 📊
   一个直观强大的矩阵图表，横跨各大核心产业（如：云计算、金融科技、电子商务、人工智能、具身智能等），深入分析各个核心岗位（如：产品经理、开发、测试、运维、设计、运营等）在 AI 浪潮下可能面临的被自动化风险，并提供转型方向指引。

3. **专属 AI 职场诊断报告 (AI Career Planning)** 🤖
   提供定制化的智能评估。用户输入当前所在的行业、岗位及相关工作内容后，系统会依据预设数据评估其工作的 AI 替代率，并自动生成结构化的应对行动和核心转型建议。

4. **咨询师可定制名片 (Editable Contact Card)** 📇
   专为职业发展咨询师及“超级个体”提供的一站式个人品牌展示板块，支持界面内部的实时修改与本地存储。

## 🛠️ 技术栈

该站点主要使用了原生技术栈，遵循轻量级、无依赖和高度可定制化的原则，并注重极致的 UI/UX 美学：
- **HTML5**：底层页面结构与语义化标签
- **CSS3 (Vanilla)**：极致美学的样式系统统筹，包括：粒子动画、玻璃拟态效应 (Glassmorphism)、暗黑现代皮肤 (Dark mode)、以及全响应式栅格/Flexbox布局。
- **JavaScript (Vanilla)**：平滑的页面滚动动画控制、动态数字自增交互、侧边栏折叠响应、热力图联动以及数据分析诊断逻辑控制。

## 🚀 本地运行部署

该项目为纯静态单页应用 (SPA)，无需复杂的构建步骤：

1. 克隆本项目代码到本地：
```bash
git clone https://github.com/hny0001/ai-career-nav.git
```
2. 直接通过浏览器打开 `index.html` 或者 `index - pm.html` 即可访问与浏览。
3. 若您需要测试本地的网络请求效果（如定制页的模拟提单），可以使用任何本地静态服务器（如 VSCode 的 Live Server 或 Python的 `Super Simple Server`）开启。
